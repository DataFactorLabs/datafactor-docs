# Core Concepts

## Reasoning-Time Advertising
Traditional ad formats break AI experiences. DataFactor's format surfaces 
relevant advertiser context as a useful data point — not a pop-up, 
not a sponsored row, not an interstitial.

## The Intent Layer
Users prompting AI with commercially meaningful queries — comparing CRMs, 
evaluating analytics stacks, researching tools — carry denser, more actionable 
intent signals than any search keyword or social impression can capture.

## Publisher-First Design
DataFactor's model keeps organic AI outputs untouched. Publishers retain 
product integrity. Users get recommendations only when contextually relevant. 
Advertisers reach users mid-decision, not mid-scroll.
